# sendgrid-export
Export data from SendGrid mail activity site to csv. To use, add a bookmark with this as the URL

``javascript: (function () { var jsCode = document.createElement('script'); jsCode.setAttribute('src', 'https://cdn.rawgit.com/andrelandry/sendgrid-export/master/bookmarklet.js'); document.body.appendChild(jsCode); }());``

Based on [this project](https://bitbucket.org/brettzink/sendgrid-activity-bookmarklet) by Brett Zink.
